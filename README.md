# Simple Interest Calculator

This is a simple interest calculator that computes interest based on principal, rate, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script `simple-interest.sh` and follow the prompts.

## Example
If Principal = 1000, Rate = 5%, Time = 2 years
Simple Interest = (1000 × 5 × 2) / 100 = 100
